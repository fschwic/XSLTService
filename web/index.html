XSLT Service

You may send the URLs of an XML and an XSLT file to this service in order to get the XML transformed by the XSLT as response.

For Example: http://.../xslt?xml=http://localhost/pathTo/my.xml&xslt=http://localhost/pathTo/my.xsl

Note: This service feels for some hostnames. If you send URLs which haven't one of that hostnames your request is delayed for 5 or more seconds.

Usage

xslt
Transform XML by XSL. Params:
xml
URL of a XML document to be transformed.
xslt
URL of a XSL document containing transformation instructions.
Several arbitrary params which are passed to the transformation (see xsl:param).
reset
Clear Transformer Pool. This resource is protected. By default the runtime containers role Admin is required.
