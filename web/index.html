XSLT Service

You may send the URLs of an XML and an XSLT file to this service in order to get the XML transformed by the XSLT as response.

For Example: http://.../xslt?xml=http://localhost/pathTo/my.xml&xslt=http://localhost/pathTo/my.xsl

Note: This service feel for some hostnames. If you send URLs which have not one of that hostnames your request is delayed for 5 seconds.
